    04DOM.html

        文件物件模型（Document Object Model, DOM）是 HTML、XML 和 SVG 文件的程式介面。
        它提供了一個文件（樹）的結構化表示法，並定義讓程式可以存取並改變文件架構、風格和內容的方法。
        DOM 提供了文件以擁有屬性與函式的節點與物件組成的結構化表示。
        節點也可以附加事件處理程序，一旦觸發事件就會執行處理程序。 
        本質上，它將網頁與腳本或程式語言連結在一起。
